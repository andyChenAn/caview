# caview
一个基于Vue2.x的PC端组件库，参考了目前主流的Vue组件库来实现。

#### 目前已经完成的组件

- [x] Message 消息提示组件
- [x] Notification 通知提醒组件
- [x] Modal 对话框组件
- [x] Drawer 抽屉组件
- [x] Popconfirm 气泡确认框组件
- [x] Loading 加载中组件
- [x] Avatar 头像组件
- [x] Badge 徽标数组件
- [x] Calendar 日历组件
- [x] Carousel 走马灯组件
- [x] Collapse 折叠面板组件
- [x] List 列表组件
- [x] Popover 气泡卡片组件
- [x] Card 卡片组件
- [x] Statistic 统计数值组件
- [x] Countdown 计数器组件
- [x] Tabs 标签页组件
- [x] Timeline 时间轴组件
- [x] Cascader 级联组件
